# Kanji Slideshow  
Automatic learning kanji through wallpaper slideshow.

# Required packages
This package requires the following Python packages:
- Pillow
- playwright

# Generate images  

```bash
python2 generate_kanji_images.py kanji_n2.csv # for N2
```
